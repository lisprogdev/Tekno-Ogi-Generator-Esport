Cara Penggunaan

1

Konfigurasi Grup

Atur jumlah grup dan jumlah tim per grup sesuai kebutuhan turnamen Anda. Pilih metode penempatan tim yang diinginkan.

2

Input Tim

Masukkan daftar tim peserta. Untuk tim unggulan, tambahkan kode [S1], [S2], dst. setelah nama tim.

3

Generate & Export

Klik tombol Generate untuk membuat grup. Hasil dapat diekspor ke Excel, PDF, atau langsung dicetak.

Pertanyaan Umum

Untuk menentukan tim unggulan, tambahkan kode [S1], [S2], dst. setelah nama tim saat input massal, atau pilih opsi "Seeded" dari dropdown saat input individual. Tim unggulan akan ditempatkan di grup berbeda jika memungkinkan.

Setelah grup digenerate, Anda dapat menggunakan tombol Excel untuk mengunduh file Excel atau tombol Print untuk mencetak hasil pembagian grup. Kedua format tersebut sudah dilengkapi dengan logo dan nama website Tekno Ogi.

Ya, jika Anda memilih metode penempatan manual, Anda dapat mengubah penempatan tim dengan drag and drop setelah grup digenerate. Untuk metode acak dan unggulan, Anda dapat mengklik tombol Generate lagi untuk hasil yang berbeda.

Tidak, semua data diproses secara lokal di browser Anda dan tidak dikirim ke server manapun. Ini berarti Anda dapat menggunakan tool ini tanpa koneksi internet setelah halaman dimuat.

Tool ini dapat menangani hingga 256 tim (16 grup dengan 16 tim per grup). Namun, untuk pengalaman terbaik, kami menyarankan tidak lebih dari 128 tim.

Jika Anda merasa terbantu dengan tool ini, Anda dapat memberikan dukungan melalui donasi di Saweria. Dukungan Anda akan membantu kami mengembangkan fitur-fitur baru dan meningkatkan kualitas layanan.
