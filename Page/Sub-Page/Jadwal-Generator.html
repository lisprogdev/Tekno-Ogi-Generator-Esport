Pertanyaan Umum

Sistem secara otomatis menghitung jadwal pertandingan berdasarkan durasi rata-rata setiap game yang dipilih, ditambah dengan waktu sterilisasi yang Anda tentukan. Jadwal akan dibuat secara berurutan dari waktu mulai hingga waktu selesai yang Anda tentukan.

Setelah jadwal digenerate, Anda dapat menggunakan tombol Excel untuk mengunduh file Excel atau tombol Print untuk mencetak hasil jadwal pertandingan. Kedua format tersebut sudah dilengkapi dengan logo dan nama website Tekno Ogi.

Saat ini, durasi game sudah ditentukan berdasarkan rata-rata waktu pertandingan untuk setiap jenis game. Namun, Anda dapat menyesuaikan waktu sterilisasi (waktu jeda antar pertandingan) sesuai kebutuhan turnamen Anda.

Tidak, semua data diproses secara lokal di browser Anda dan tidak dikirim ke server manapun. Ini berarti Anda dapat menggunakan tool ini tanpa koneksi internet setelah halaman dimuat.

Aplikasi ini dapat menangani hingga 500 pertandingan dalam satu jadwal. Namun, untuk pengalaman terbaik, kami menyarankan membuat jadwal dengan rentang waktu yang masuk akal sesuai dengan jumlah pertandingan yang direncanakan.

Jika Anda merasa terbantu dengan tool ini, Anda dapat memberikan dukungan melalui donasi di Saweria. Dukungan Anda akan membantu kami mengembangkan fitur-fitur baru dan meningkatkan kualitas layanan.
